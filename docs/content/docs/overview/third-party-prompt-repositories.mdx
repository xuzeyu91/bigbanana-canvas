# 提示词库数据源

提示词中心不再聚合 GitHub 提示词仓库，统一使用 [GPT-Image-2 Prompt Gallery](https://image2.tree456.com/) 的公开案例数据。

| 数据入口 | 用途 |
| --- | --- |
| `https://image2.tree456.com/data/ingested_tweets.json` | 标题、完整提示词、分类、原始案例链接与时间 |
| `https://image2.tree456.com/data/webp-manifest.json` | 可用图片目录与 WebP 缩略图目录 |

服务端只保留同时具备完整提示词和可用封面的案例，并在内存中缓存一小时。`bigbanana.tree456.com` 的生图工作台会跳转到该提示词库，因此不重复抓取另一份数据。
